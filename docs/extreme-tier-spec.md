# Extreme Tier — Authoring Spec (research-based)

Calibration from real Bluebook hard Module 2s: back third is top-difficulty
(Math ~6–8/22, RW ~7–9/27); only ~3–5 per module are "brutal" (<40% accuracy).
Extreme sets mix ~50% hard / ~30% top-tier / ~20% brutal. Key signature:
students answer brutal questions WRONG AT NORMAL SPEED — traps produce
confident wrong answers, not stalls.

R&W constructions: second-meaning common words as keys (economy=efficiency,
wanting=deficient); double-contrast blanks; 19th-c literary excerpts with
irony/free indirect discourse; hedged-science support/weaken with
true-but-irrelevant and supports-the-wrong-claim distractors; cross-text
agrees-with-data-rejects-inference; stacked interrupters (appositive within
appositive); inverted/long-delayed subjects; reduced relative clauses;
restrictive vs nonrestrictive titles/citations; adjacent-category transitions
(therefore vs in addition, specifically vs for example); concession-then-return
pivots; dual-constraint synthesis goals.

Math constructions: parameter systems with fractions (no/infinite solutions,
fractional k); wordy linear models with unit traps (per-minute vs per-hour,
"years after 2015" intercept trap); |ax−b|=k solution counts; parametric
discriminants (tangency → b²−4ac=0, fractional c); fractional/negative exponent
manipulation (x·⁵√(x³·√x) → x^(a/b), grid a+b); composition/transformation
combos g(x)=f(x−3)+2; exponential models a(0.88)^(t/5) with (0.88)^(5t) traps;
three-way conditional probability from two-way tables; weighted means with
unknown n; margin-of-error inference limits; circle equations with leading
coefficient ≠1 + tangency conditions; similar-triangle chains
(altitude-to-hypotenuse); complementary-angle trig; volume+density/rate with
buried unit conversions.

Time-sink traps by design: shortcut-vs-grind (structure or Desmos solves in
30s); intermediate-value distractors (x when asked 2x+1); sign slips;
true-statement-wrong-question.

In-app encoding: set level:'Extreme' (display badge); every question
difficulty:'Difícil' + extreme:true (mock assembler: tier 'extreme' draws
extreme-flagged first, tops up with regular Difícil). Extreme Mocks seeds 400+N.
